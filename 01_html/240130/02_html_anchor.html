앵커태그

정의

앵커(a) 태그는 링크를 만들어주는 태그입니다.

속성

  • href: 링크할 주소를 지정
  • target: 링크를 열 위치를 지정

href 속성

  • 링크할 주소를 지정한다
  • 상대경로와 절대 경로 모두 사용가능
  • 다른 페이지로 이동할수있게한다
  • 페이지 내부에서 특정위치로 이동가능
  • 이메일과 전화번호도 연결 가능한다

타겟 속성

  • 링크를 열 위치를 지정한다
  • 기본값은_self이다.
  • _self는 현재 창에서 링크를 연다
  • _blank는 새창에서 링크를 연다

target="_blank"보안 문제

  • target="_blank" 보안 취약점(정보노출)이 존재할수도잇다
  • 브라우저의 팝업 차단기에 의해 차단 될 우려가 있다
  • target="_blank"에 rel="noopener noreferrer"을 추가하면, 외부페이지에서 현재페이지에 대한 악의적인 조작을 방지하고, 사용자의 개인정보를 보호할수있어서 보다 안전하게 처리된다
noopener
새창,새탭을 열때,부모창에대한 참조를 막는다
새로운 창이 부모창에대한 제어권을 가질수없게한다.(보안문제제거)
다른페이지의 스크립트가 새로열린 창의 내용을 조작할수없게한다.
noreferrer
http 리퍼러 헤더를 제거한다
리퍼러 헤더는 현재 페이지에서 새로 열린 페이지로 전달되는 정보이다 noreferrer를 쓰면, 외부 사이트가 어디에서 링크를 클릭했는지 알수없게한다
