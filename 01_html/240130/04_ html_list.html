목록list태그

(ol.li),(ul,li),(dl,dt,dd)는 목록을 만드는데 사용된다.

(ol.li)

번호가 붙은 목록을 만드는데 사용한다.

  1. ol(odered list)-순서가있는 목록
  2. li(list item)-목록항목

(ul,li)

점으로 나타나는 목록을 만드는데 사용한다.

  • ul(unodered list)-순서가없는목록
  • li(list item)-목록항목

(dl,dt,dd)

용어와 해당 설명을 정의하는데 사용한다.

dl(descrtion list)
설명목록
dt(descrtion term)
설명용어
dd(descrtion definition)
설명에대한 정의를 의미합니다
