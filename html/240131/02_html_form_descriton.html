입력양식 태그

fieldset

입력양식을 그룹으로 묶어주는 태그(disabled)

label

입력양식의 설명을 만들어주는 태그(for:입력양식의id 와 연결)

form 태그

form태그는 입력양식폼을 만들어주는 태그이다

form태그의 속성

  • action: 입력 데이터를 전송할 서버의 주소 지정(상대경로,절대경로 가능)/li>
  • method: 입력데이터를 전송할 방식을 지정 (기본값:get/post)

input 태그

input 태그의 속성

  • type

    입력 양식의 종류를 지정(text, password, number, range, color, email, tel, url, search, date, time, week, datetime-local, checkbox, radio, file, image, submit, reset, button, ...)

  • name

    입력한 데이터를 전송하기 위해 지정(값에 이름을 붙인다 [name=value])

  • value

    입력 양식의 값을 의미하고, 지정하면 기본값으로 설절된다.

  • placeholder,required,readonly,disbied....

    입력양식의 힌트,필수여부,읽기전용설정, 사용불가 설정등을 할수있다.

select

여러개의 선택지를 가진 입력양식을 생성

option

select,datelist 태그의 선택지를 만들어주는 태그(select, name,value,disabled)

textarea

여러줄의 텍스트를 입력할수있는 입력 양식(cols,rows 속성으로 크기를 설정)

button

버튼을 생성(type,name,disabled),빈태그가 아니다
